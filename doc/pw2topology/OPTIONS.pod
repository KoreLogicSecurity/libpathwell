
=head1 OPTIONS

=over 4

=item B<[-f {file|-}]>

Specifies the name of the input file. A value of '-' will cause the
program to read from stdin.

=item B<[-s token-set]>

Specifies the ID (see below) of the token set to use while performing
conversions. The default value is '1'.

    1 - ?d, ?l, ?u, ?s
    2 - ?d, ?l, ?u, ?s, ?f
    3 - ?d, ?l, ?u, ?s, ?f, ?x
    4 - ?d, ?l, ?u, ?s, ?f, ?x, ?H

=back
